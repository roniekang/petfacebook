# Pettopia - 서비스 컨셉 기획서

## 서비스 개요

**Pettopia**는 반려동물(펫)을 중심으로 한 소셜 네트워크 서비스입니다.
기존 페이스북의 소셜 구조를 펫 중심으로 재구성하여, 반려동물과 보호자 간의 연결, 지역 커뮤니티, 생활 서비스를 통합 제공합니다.

### 핵심 철학

> **펫이 주인공입니다.**
> - 계정의 중심은 펫 (Pet)이며, 집사(보호자)는 펫의 대리인
> - 집사는 1개의 펫에 대해 하나의 계정을 생성 (여러 펫 → 여러 계정)
> - 모든 소셜 활동(친구, 피드, 커뮤니티)은 펫 단위로 이루어짐
> - 지역 기반 서비스 (Geo IP / Geo Tag)

## Phase 1 - MVP 기능 정의

### 1. 인증 / 계정

| 기능 | 설명 |
|------|------|
| 집사 회원가입 / 로그인 | 이메일, 소셜 로그인 지원 |
| 펫 계정 생성 | 집사가 펫 정보를 입력하여 펫 계정 생성 (1펫 = 1계정) |
| 펫 계정 전환 | 여러 펫을 키우는 집사는 계정 간 전환 가능 |
| Geo IP 기반 지역 설정 | 위치를 자동 감지하여 지역 커뮤니티 연결 |

### 2. 펫 프로필

| 기능 | 설명 |
|------|------|
| 펫 프로필 | 이름, 종류, 나이, 사진 등 기본 정보 (=계정 프로필) |
| 펫 활동 기록 | 산책, 놀이 등 일상 활동 기록 (Geo IP 기반) |
| 펫 관리 | 건강 기록, 일정 관리 등 |

### 3. 피드 / 콘텐츠

| 기능 | 설명 |
|------|------|
| 글쓰기 (포스트) | 텍스트 기반 게시글 작성 |
| 사진 업로드 | 펫 사진 업로드 및 공유 |
| 스토리 | 배경/비디오에 물리 효과 적용, 사진→동적 영상 변환, Geo Tag 기반 분류 |
| 검색 | 펫, 친구, 지역, 콘텐츠 통합 검색 |

### 4. 소셜 / 커뮤니티

| 기능 | 설명 |
|------|------|
| 펫 친구 시스템 | 집사가 폰 앱을 통해 펫의 친구를 추가 (QR코드, 근처 검색, 지역 추천) |
| 지역 커뮤니티 | 같은 지역 내 그룹(우리) 생성 및 소통 |
| 산책 모임 | 오프라인 산책 모임 매칭 및 관리 |

### 5. 생활 서비스

| 기능 | 설명 |
|------|------|
| 동물병원 | 지역 기반 동물병원 정보 제공 및 연동 |
| 펫 보험 | 펫 보험 상품 정보 제공 및 연동 |

### 6. Pet Haven (펫 천국 - 추모 공간)

| 기능 | 설명 |
|------|------|
| 메모리얼 공간 | 무지개 다리를 건넌 펫을 추모하는 디지털 공간 |
| "살아있는 것처럼" | 사진/영상을 활용해 마치 살아있는 것처럼 기억하는 감성 기능 |
| Happy 메모리 | 행복했던 순간들을 모아 보여주는 기능 |
| 조의금 시스템 | 친구 펫이 무지개 다리를 건넌 경우, 조의금을 보낼 수 있음 |
| 추모관 꾸미기 | 조의금은 Pettopia 인앱 화폐로 전환되어 추모관 꾸미기에 사용 가능 |

### 7. 인앱 화폐 시스템

| 기능 | 설명 |
|------|------|
| Pettopia 코인 | 서비스 내 화폐 단위 |
| 조의금 전환 | 조의금 → Pettopia 코인으로 자동 전환 |
| 추모관 꾸미기 | 코인으로 추모관 배경, 장식, 꽃, 촛불 등 아이템 구매 |
| 향후 확장 | 다른 서비스(프리미엄 스토리 효과 등)에도 활용 가능 |

## 핵심 차별화 포인트

1. **펫 = 계정** - 사람이 아닌 펫이 소셜 활동의 주체 (집사는 1펫 1계정 생성)
2. **폰 앱 기반 친구 추가** - 집사들끼리 만나서 펫의 친구를 바로 추가 (QR, 근처 검색)
3. **Geo 기반 연결** - Geo IP / Geo Tag를 활용한 지역 기반 커뮤니티
4. **온/오프라인 연결** - 온라인 소통뿐 아니라 산책 모임 등 오프라인 활동 연계
5. **Pet Haven + 조의금** - 감성적 추모 공간 + 조의금 기반 인앱 화폐로 추모관 꾸미기
6. **생활 서비스 통합** - 병원, 보험 등 펫 생활에 필요한 서비스 연동

## 계정 모델

```
집사 (Guardian)
├── 로그인/인증 주체
├── 펫 계정 1: 뽀삐 (DOG) ← 모든 소셜 활동의 주체
├── 펫 계정 2: 나비 (CAT) ← 별도의 독립적인 계정
└── 계정 간 전환 가능
```

## 관계 모델

```
[펫 계정 A: 뽀삐] ──친구──▶ [펫 계정 B: 초코]
       │                          │
       │ (집사가 대리 운영)          │ (집사가 대리 운영)
       │                          │
   집사 A                      집사 B
   (폰 앱으로 친구 추가)         (수락/거절)
```

```
Pet Haven 흐름:
[펫 계정] ──사망──▶ [Pet Haven 전환]
                        │
                  ┌─────┼─────────────┐
                  │     │             │
             추모 페이지  Happy 메모리   추모관 꾸미기
                  │                     ▲
            친구들이 조의금 전송 ──▶ Pettopia 코인
```
